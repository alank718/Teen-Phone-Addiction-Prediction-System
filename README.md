# Teen Phone Addiction Prediction System

## 📌 Introduction
The Teen Phone Addiction Prediction System is a Machine Learning-based project developed to analyze teenagers' smartphone usage habits and predict their level of phone addiction.

With the rapid increase in screen time and social media dependency among teenagers, excessive phone usage has become a serious issue affecting mental health, sleep quality, academic performance, and physical activity. This project aims to identify addiction levels at an early stage and provide corrective suggestions for healthier digital habits.

The system takes various behavioral inputs such as screen time, gaming hours, sleep duration, social media usage, and study hours to predict whether the addiction level is **Low, Moderate, or High**.

In addition to prediction, the project also provides personalized recommendations to help users reduce phone dependency and maintain a balanced lifestyle.
